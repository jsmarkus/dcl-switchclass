# dcl-class-switch

DCL util to bind attribute to set of CSS classes

## Getting Started
Install the module with: `npm install dcl-class-switch`

```javascript
var dcl_class_switch = require('dcl-class-switch');
dcl_class_switch.awesome(); // "awesome"
```
